# Pixellab Web - محرر الرسومات

تطبيق ويب قوي وسهل الاستخدام لتصميم الرسومات والصور، مستوحى من تطبيق Pixellab Plus. يوفر مجموعة شاملة من الأدوات لإنشاء تصاميم احترافية مباشرة من متصفحك.

## المميزات الرئيسية

### أدوات النصوص
- إضافة وتحرير النصوص بسهولة
- دعم خطوط متعددة (Arial, Georgia, Times New Roman, Courier New, Verdana)
- التحكم الكامل في حجم الخط واللون
- تأثيرات متقدمة للنصوص (ظل، حد، خلفية)

### الأشكال والعناصر
- إضافة أشكال أساسية: مستطيل، دائرة، مثلث
- التحكم الكامل في الألوان والحدود
- إضافة الصور من جهازك
- تحجيم وتدوير العناصر بسهولة

### أدوات التحرير
- نظام الطبقات المتقدم
- التراجع والإعادة (Undo/Redo)
- نسخ ولصق العناصر
- حذف العناصر المحددة
- التحكم في الشفافية والدوران

### التصدير والحفظ
- تصدير التصاميم كصور PNG أو JPG بجودة عالية
- حفظ المشاريع محلياً (Local Storage)
- استعادة المشاريع المحفوظة بسهولة

## المتطلبات

- متصفح حديث يدعم HTML5 Canvas
- Node.js و npm/pnpm

## التثبيت والتشغيل

### 1. استنساخ المشروع
```bash
git clone https://github.com/yourusername/pixellab_web.git
cd pixellab_web
```

### 2. تثبيت المكتبات
```bash
pnpm install
```

### 3. تشغيل خادم التطوير
```bash
pnpm dev
```

سيتم فتح التطبيق تلقائياً على `http://localhost:3000`

## الاستخدام

### إضافة نص
1. أدخل النص المطلوب في حقل "أدخل النص"
2. اختر الخط والحجم واللون
3. انقر على "إضافة نص"

### إضافة أشكال
1. اختر لون التعبئة والحد
2. انقر على الشكل المطلوب (مستطيل، دائرة، مثلث)
3. سيظهر الشكل على لوحة الرسم

### إضافة صور
1. انقر على زر "صورة"
2. اختر صورة من جهازك
3. سيتم إضافة الصورة إلى لوحة الرسم

### تحرير العناصر
1. انقر على العنصر لتحديده
2. استخدم الخصائص في الجانب الأيمن للتحكم في:
   - الحجم والارتفاع
   - الدوران
   - الشفافية
   - الألوان

### حفظ وتصدير
- **حفظ المشروع**: استخدم زر "المشاريع" لحفظ عملك
- **تصدير كصورة**: انقر على "PNG" أو "JPG" لتنزيل التصميم

## البنية التقنية

### المكتبات المستخدمة
- **React 19**: لبناء الواجهة
- **Fabric.js**: محرك الرسومات المتقدم
- **Tailwind CSS**: للتصميم والأسلوب
- **shadcn/ui**: مكونات الواجهة الجاهزة
- **Lucide React**: الأيقونات
